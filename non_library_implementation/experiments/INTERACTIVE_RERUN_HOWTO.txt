INTERACTIVE RERUN MODE AND CUSTOM CLI GUIDE

This guide explains:
1. how to use the end-of-run INTERACTIVE RE-RUN MODE prompt,
2. what to type at the prompt,
3. what to type directly in the CLI,
4. which scene/module each algorithm contains,
5. which keys control each scene/module.

The examples below are written in neutral form with:
python "Theory Algorithms/<script>.py"

GENERAL USAGE

Default run, then prompt:
python "Theory Algorithms/<script>.py"

When the terminal shows:
INTERACTIVE RE-RUN MODE
Custom parameters:
type one comma-separated line such as:
key1=value1, key2=value2, key3=value3

Direct one-shot CLI custom run:
python "Theory Algorithms/<script>.py" key1=value1 key2=value2 key3=value3

VALUE SYNTAX

Booleans:
True
False

Lists:
[1,2,3]
[0.1,0.2,0.3]
['0101','1110']

Strings:
'output_name'

Null:
None
